# CuPER - Custom ParsER
Custom parsing module written in Python.


- **Warning**: Use this library at your own risk.
    + Reason 1. It is under development.
    + Reason 2. Python is slow, so this library is too.

### Getting started
> Python version: 3.8+

- Linux/Mac
```console
$ cd path/to/cuper
$ python3 example.py
the magic of python
```

- Windows
```console
$ cd path/to/cuper
$ python example.py
the magic of python
```
